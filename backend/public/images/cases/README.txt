ПАПКА ДЛЯ КАРТИНОК КЕЙСОВ
========================

Вставьте сюда картинки кейсов.

Рекомендуемые имена файлов:
- free_case.png (или .jpg) - для ежедневного кейса
- newyear_case.png - для новогоднего кейса
- crypto_case.png - для крипто кейса
- onlynft_case.png - для NFT кейса
- genesis_case.png - для генезис кейса
- elite_case.png - для элитного кейса

Формат: PNG или JPG
Рекомендуемый размер: 200x200px или больше (квадратные изображения)

После добавления картинок, обновите HTML файл, чтобы использовать эти изображения вместо эмодзи.
